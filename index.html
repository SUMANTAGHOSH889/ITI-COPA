COPA Theory Mock Paper

Interactive Version

Q1. Which of the following is an input device?

Answer: B) Keyboard

Q2. Which MS Office program is used to create slideshows?

Answer: C) PowerPoint

Q3. What does CPU stand for?

Answer: B) Central Processing Unit

Q4. HTML is used for:

Answer: B) Web page creation

Q5. Which of the following is cloud storage?

Answer: A) Google Drive

Q6. Shortcut key to copy text in Windows?

Answer: C) Ctrl + C

Q7. Which of the following is NOT an operating system?

Answer: C) Chrome

Q8. What is the default file extension of MS Word?

Answer: A) .docx

Q9. CMD command to list all files in a directory?

Answer: A) dir

Q10. Internet Explorer is an example of:

Answer: A) Browser

Q11. Explain the difference between RAM and ROM.

Answer: RAM is volatile memory used for temporary storage while ROM is non-volatile memory storing permanent data.

Q12. Write the steps to create a table in MS Word.

Answer: Insert → Table → Select rows and columns → Enter data → Format as required.

Q13. How do you create a simple formula in MS Excel?

Answer: Select cell → Type '=' → Enter formula (e.g., =SUM(A1:A5)) → Press Enter.

Q14. Explain “CC” and “BCC” in email.

Answer: CC = Carbon Copy, visible to all; BCC = Blind Carbon Copy, hidden recipients.

Q15. Write 3 advantages of using cloud storage.

Answer: Accessible anywhere, backup and recovery, cost-effective storage.

Q16. _______ is the brain of the computer.

Answer: CPU

Q17. The extension of a PowerPoint file is _______.

Answer: .pptx

Q18. _______ is used to create web pages.

Answer: HTML

Q19. Ctrl + Z is used for _______.

Answer: Undo

Q20. Google Docs is an example of _______.

Answer: Cloud application
